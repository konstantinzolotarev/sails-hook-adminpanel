# Sections Dropdown

Is a dropdown in top-left screen side, for fast switching between sections


```javascript
module.exports.adminpanel = {
    sections: [
        {
            key: 'users',
            title: 'Users'
            link: '/users'
        },
        {
            key: 'content',
            title: 'Content items'
        }
    ]
```

> Dashboard by default added to this dropdown
